# AttendanceLotus
is an attendance system

## Features
#### Admin
- Add teachers and students
- Add courses, classes, and subjects

#### Teacher
- Add classes
- Add students to classes
- Take attendance
- View attendance

## Setup
1. open <https://localhost/phpmyadmin>
2. create database
3. click import
4. Locate `attendanceLotus.sql` or drag it to the window
